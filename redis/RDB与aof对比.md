RDB与aof对比

| 命令       | RDB          | AOF          |
| ---------- | ------------ | ------------ |
| 启动优先级 | 低           | 高           |
| 体积       | 小           | 大           |
| 恢复速度   | 快           | 慢           |
| 数据安全性 | 容易丢失数据 | 根据策略决定 |

生产环境都可以启用，redis启用的时候如果既有RDB文件，又有aof文件，则优先选择aof文件恢复，因为aof一般来说数据更安全一点。

#### Redis 混合持久化

aof-use-rdb-preamble yes
